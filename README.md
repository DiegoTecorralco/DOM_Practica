# DOM_Practica
Aqui tendremos todo lo que aprendimos de DOM aprendido en la clase de Desarrollo Web Profesional

- Aprenderemos a seleccionar, crear, modificar y eliminar elementos del DOM de forma interactiva.

## Objetivo
- **Seleccionar elementos** por metodos como getByElement
- **Cambiar textos, atributos y estilos de forma dinamica** mediante propiedades como textContent y classList
- **Crear y eliminar nodos** generando nuevos elementos HTML con createElement y eliminarlos con removeChild, 
- **Gestionar eventos** Responder a clics, escritura en teclado y otras interacciones del usuario mediante addEventListener
